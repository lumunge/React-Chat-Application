### WOZZUP

### ABOUT

A simple chat application using react chat engine with react features and chet
engine backend capabilities

### TECHNOLOGIES USED:

    * HTML5
    * CSS3
    * Reactjs
    * Chatengine API
    * Axios

### FEATURES

    * Support group chats, one on one chats.
